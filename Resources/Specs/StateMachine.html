StateMachine

The main class for managing GUI views and menu view states.

A. Chang, 5/15




CLASS StateMachine

No superclass.

State variables:

State * activeState
The activeState value indicates the currently selected state.

std::vector stateArray
A vector collection of States that represent individual views.

int numStates
The numStates value stores the total number of states accessible by the state machine.

Constructors:

StateMachine
None.

No arguments.

State change:
None.

Destructor:

Methods:

registerState
Creates a new state and stores it.

Arguments:

State * st
The state to be added to the stateArray.

State change:
Adds the passed state to the stateArray vector collection.

Return value:
None.

transition
Exits current state and calls new one.

Arguments:

string stateName
A string value of the new state.

State change:
Changes the activeState to the state with the passed stateName.

Return value:
None.

destroy

No arguments.
